# Supported Networks

ZeroDev supports EVM networks including:

- Arbitrum One
  - Mainnet
  - Sepolia
- Avalanche
  - Mainnet
  - Fuji
- Base
  - Mainnet
  - Sepolia
- Binance Smart Chain
  - Mainnet
- Ethereum
  - Mainnet
  - Sepolia
- Optimism
  - Mainnet
  - Sepolia
- Polygon
  - Mainnet
  - Mumbai
- Linea
  - Mainnet
  - Testnet
- opBNB
  - Mainnet
- Astar Zkatana
  - Testnet

If you don't see a EVM network that you'd like support for, feel free to reach out to us on [Discord](https://discord.gg/KS9MRaTSjx).
